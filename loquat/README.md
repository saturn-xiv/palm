# USAGE

- For x86_64

  ```bash
  xmake f -p linux -a x86_64 -m release
  ```

- For aarch64

  ```bash
  xmake f -p cross -m release --sdk=$HOME/local/arm-gnu-toolchain-13.2.Rel1-x86_64-aarch64-none-linux-gnu
  ```
