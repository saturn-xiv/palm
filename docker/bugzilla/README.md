# USAGE

```bash
$ ./start.sh
> cd /var/www/bugzilla/
# vim /etc/bugzilla/setup.pl
> ./checksetup.pl /etc/bugzilla/setup.pl
# https://<servername>/bugzilla/
> /start.sh
```
